## 🎮 使用方法

### **完整模式 (推荐首次使用)**
```bash

# 或
./github-push-ultimate.sh full
```

**适用场景**：
- 首次推送项目
- 遇到复杂问题时
- 需要完整配置时

### **快速模式 (日常使用)**
```bash

# 简化写法

```

**适用场景**：
- 日常代码更新

## 🛡️ 错误处理机制

### **仓库不存在**
```
❌ 错误: Repository not found
🔧 解决方案:
1) 立即创建仓库
2) 使用不同名称  
3) 强制创建并推送
```

### **权限被拒绝**
```
❌ 错误: Permission denied
🔧 解决方案:
1) 重新设置SSH密钥
2) 检查SSH配置
3) 使用HTTPS方式
```

### **网络连接问题**
```
❌ 错误: Connection timed out
🔧 解决方案:
1) 重试连接
2) 检查网络
3) 使用代理
```

### **认证失败**
```
❌ 错误: Authentication failed
🔧 解决方案:
1) 重新设置SSH密钥
2) 使用GitHub Token
3) 重新认证
```

### **远程拒绝推送**
```
❌ 错误: remote rejected
🔧 解决方案:
1) 强制推送
2) 拉取后重试
3) 重置远程分支
```
