## 🎯 统一的样式标准

### 📐 布局结构
```html
<div class="bg-white min-h-screen">
    <div class="px-6 pt-6 mb-8">
        <h1 class="text-2xl font-bold text-gray-900">页面标题</h1>
        <p class="text-sm text-gray-600 mt-2">页面描述</p>
    </div>
    
    <div class="px-6">
        <!-- 页面内容 -->
    </div>
</div>
```

### 🎨 样式特点
- **外层容器**: `bg-white min-h-screen` - 白色背景，最小高度为屏幕高度
- **标题区域**: `px-6 pt-6 mb-8` - 左右内边距6，顶部内边距6，底部外边距8
- **主标题**: `text-2xl font-bold text-gray-900` - 2xl字体大小，粗体，深灰色
- **副标题**: `text-sm text-gray-600 mt-2` - 小字体，中灰色，顶部外边距2
- **内容区域**: `px-6` - 左右内边距6

```html

    <div class="px-6 pt-6 mb-8">

        <p class="text-sm text-gray-600 mt-2">描述</p>
    </div>
    
    <div class="px-6">
        <!-- 内容 -->
    </div>
</div>
```

1. **外层容器**: 从 `container mx-auto px-4 py-8` 改为 `bg-white min-h-screen`
2. **标题区域**: 从 `mb-8` 改为 `px-6 pt-6 mb-8`
3. **主标题**: 从 `text-3xl` 改为 `text-2xl`，移除 `mb-2`
4. **副标题**: 添加 `text-sm` 和 `mt-2`
5. **内容区域**: 添加 `px-6` 包装器
