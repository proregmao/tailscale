# 05-Tailscale命令行工具

## 工具概述

Tailscale提供了两个主要的命令行工具：
- **tailscale**: 用户交互的客户端工具
- **tailscaled**: 后台运行的守护进程

## tailscale客户端命令

### 1. 基础命令

#### 连接管理
```bash
# 启动并连接到网络
tailscale up [flags]

# 断开连接
tailscale down

# 登录到Tailscale
tailscale login

# 登出
tailscale logout

# 切换账户
tailscale switch <account>
```

#### 状态查询
```bash
# 查看连接状态
tailscale status [--json] [--peers] [--self]

# 查看IP地址
tailscale ip [-4] [-6]

# 查看网络检查
tailscale netcheck

# 查看版本信息
tailscale version [--json]
```

### 2. 网络诊断

#### Ping测试
```bash
# Ping其他节点
tailscale ping <hostname-or-ip>

# 详细输出
tailscale ping --verbose <target>

# TSMP级别ping
tailscale ping --tsmp <target>

# ICMP级别ping
tailscale ping --icmp <target>

# PeerAPI测试
tailscale ping --peerapi <target>

# 限制ping次数
tailscale ping -c 5 <target>
```

#### 网络连接
```bash
# 网络连接测试
tailscale nc <hostname> <port>

# 监听模式
tailscale nc -l <port>
```

#### WhoIs查询
```bash
# 查询IP对应的设备和用户
tailscale whois <ip>

# JSON格式输出
tailscale whois --json <ip>

# 指定协议
tailscale whois --proto tcp <ip:port>
```

### 3. 配置管理

#### 设置命令
```bash
# 设置配置选项
tailscale set [flags]

# 设置操作员用户
tailscale set --operator=<user>

# 设置主机名
tailscale set --hostname=<name>

# 设置广告路由
tailscale set --advertise-routes=<routes>

# 设置出口节点
tailscale set --exit-node=<node>

# 接受路由
tailscale set --accept-routes

# 启用SSH
tailscale set --ssh

# 设置屏蔽密钥
tailscale set --shields-up
```

#### 配置主机
```bash
# 配置主机功能
tailscale configure-host [subcommand]

# 配置Kubernetes
tailscale configure kubeconfig <cluster-name>

# Synology配置
tailscale configure synology
```

### 4. 服务功能

#### 文件共享 (Taildrop)
```bash
# 发送文件
tailscale file cp <file> <target>:

# 获取文件
tailscale file get <file>

# 列出接收的文件
tailscale file list
```

#### 服务暴露
```bash
# 启动本地服务
tailscale serve <port>

# 暴露到公网
tailscale funnel <port>

# 查看服务配置
tailscale serve status

# 重置服务配置
tailscale serve reset
```

#### SSH功能
```bash
# SSH连接
tailscale ssh <hostname>

# 指定用户
tailscale ssh <user>@<hostname>

# 指定端口
tailscale ssh -p <port> <hostname>
```

### 5. 出口节点管理

```bash
# 列出出口节点
tailscale exit-node list

# 按国家过滤
tailscale exit-node list --filter=<country>

# 建议最佳出口节点
tailscale exit-node suggest
```

### 6. 证书管理

```bash
# 获取TLS证书
tailscale cert <domain>

# 指定证书目录
tailscale cert --cert-dir=<dir> <domain>
```

### 7. 系统策略

```bash
# 列出系统策略
tailscale syspolicy list

# 查看策略详情
tailscale syspolicy show <policy>
```

### 8. 调试命令

```bash
# 调试命令集
tailscale debug [subcommand]

# 网络解析
tailscale debug resolve <hostname>

# 构建信息
tailscale debug go-buildinfo

# 中继服务器信息
tailscale debug peer-relay-servers

# 捕获网络包
tailscale debug capture

# 重新绑定网络
tailscale debug rebind

# 重新STUN
tailscale debug restun
```

### 9. 更新管理

```bash
# 检查更新
tailscale update

# 安装更新
tailscale update --yes

# 检查更新状态
tailscale update --check
```

### 10. 许可证信息

```bash
# 查看开源许可证
tailscale licenses
```

## tailscaled守护进程

### 1. 启动参数

```bash
# 基本启动
tailscaled [flags]

# 指定TUN设备
tailscaled --tun=<device>

# 用户空间网络
tailscaled --tun=userspace-networking

# 指定端口
tailscaled --port=<port>

# 指定状态文件
tailscaled --state=<file>

# 启用调试
tailscaled --debug=<addr>

# SOCKS5代理
tailscaled --socks5-server=<addr>

# HTTP代理
tailscaled --outbound-http-proxy-listen=<addr>

# 详细日志
tailscaled --verbose=<level>

# 配置文件
tailscaled --config=<file>
```

### 2. 子命令

```bash
# 清理系统状态
tailscaled --cleanup

# 安装系统服务
tailscaled install-system-daemon

# 卸载系统服务
tailscaled uninstall-system-daemon
```

## 环境变量

### 1. 常用环境变量

```bash
# 控制服务器URL
export TS_CONTROL_URL=https://your-control-server.com

# 认证密钥
export TS_AUTHKEY=your-auth-key

# 状态目录
export TS_STATE_DIR=/var/lib/tailscale

# 套接字路径
export TS_SOCKET=/var/run/tailscale/tailscaled.sock

# 日志级别
export TS_LOG_VERBOSITY=2

# 禁用日志上传
export TS_NO_LOGS_NO_SUPPORT=true
```

### 2. 容器环境变量

```bash
# 容器启动模式
export TS_USERSPACE=true

# 接受DNS
export TS_ACCEPT_DNS=true

# 额外参数
export TS_EXTRA_ARGS="--advertise-routes=10.0.0.0/8"

# Kubernetes模式
export TS_KUBE_SECRET=tailscale-auth

# 代理配置
export TS_SOCKS5_SERVER=localhost:1080
export TS_OUTBOUND_HTTP_PROXY_LISTEN=localhost:8080
```

## 使用示例

### 1. 基本使用流程

```bash
# 1. 启动守护进程
sudo tailscaled

# 2. 登录并连接
tailscale up

# 3. 查看状态
tailscale status

# 4. 测试连接
tailscale ping other-device

# 5. 断开连接
tailscale down
```

### 2. 高级配置示例

```bash
# 作为子网路由器
tailscale up --advertise-routes=192.168.1.0/24 --accept-routes

# 作为出口节点
tailscale up --advertise-exit-node

# 启用SSH访问
tailscale up --ssh

# 设置自定义主机名
tailscale up --hostname=my-server

# 使用认证密钥
tailscale up --authkey=tskey-auth-...
```

### 3. 故障排除

```bash
# 检查网络连接
tailscale netcheck

# 详细ping测试
tailscale ping --verbose target

# 查看调试信息
tailscale debug status

# 重新连接
tailscale down && tailscale up

# 查看日志
journalctl -u tailscaled -f
```
