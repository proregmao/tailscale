## 🏗️ 技术架构

### 后端服务 (Go)
- **框架**: Gin Web Framework
- **数据库**: SQLite + GORM ORM
- **协议**: 完全兼容Tailscale客户端
- **性能**: 支持大规模并发连接
- **监控**: 内置告警引擎和日志系统

### 前端界面 (Svelte)
- **框架**: Svelte + TypeScript
- **样式**: TailwindCSS + DaisyUI
- **特性**: 响应式设计，实时数据更新
- **体验**: 直观的可视化管理界面

### 核心特性
- **无限制设备**: 移除官方Tailscale的设备数量限制
- **完全自托管**: 不依赖Tailscale官方服务
- **协议兼容**: 100%兼容Tailscale客户端
- **可视化管理**: 现代化的Web管理界面
- **扩展性强**: 支持自定义DERP服务器
