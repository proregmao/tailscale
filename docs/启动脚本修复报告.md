# 🔧 启动脚本修复报告

## 🐛 问题描述

在首次运行启动脚本时遇到了前端服务启动超时的问题：

```
[2025-08-01 08:46:50] 前端服务 启动超时！
[2025-08-01 08:46:50] 前端启动失败，请检查日志: /root/tailscale/logs/frontend.log
```

## 🔍 问题分析

通过分析前端日志发现了根本原因：

### 1. **端口冲突问题**
```log
> headscale-ui@2025.07.12 dev
> vite dev --port 8080 --host 0.0.0.0

Port 8080 is in use, trying another one...
Port 8081 is in use, trying another one...

  VITE v5.4.19  ready in 1692 ms
  ➜  Local:   http://localhost:8082/
```

**问题原因**：
- 前端配置为在端口 8080 启动
- 后端已经占用了端口 8080
- Vite 自动选择了端口 8082
- 启动脚本仍在等待端口 5173，导致超时

### 2. **配置不一致**
- `headscale-ui/package.json` 中配置：`--port 8080`
- `headscale-ui/vite.config.js` 中代理指向：`http://localhost:8081`
- 启动脚本期望端口：`5173`

## ✅ 修复方案

### 1. **修复前端端口配置**

**修改 `headscale-ui/package.json`**：
```json
// 修复前
"dev": "vite dev --port 8080 --host 0.0.0.0"

// 修复后  
"dev": "vite dev --port 5173 --host 0.0.0.0"
```

**修改 `headscale-ui/vite.config.js`**：
```javascript
// 修复前
proxy: {
    '/api': {
        target: 'http://localhost:8081',
        // ...
    }
}

// 修复后
proxy: {
    '/api': {
        target: 'http://localhost:8080',
        // ...
    }
}
```

### 2. **增强启动脚本智能检测**

**Shell脚本 (`start.sh`) 改进**：
```bash
# 修复前：只检查固定端口 5173
if ! wait_for_port 5173 "前端服务"; then
    # 失败处理
fi

# 修复后：智能检测多个端口
local frontend_started=false
local max_attempts=30

while [ $attempt -le $max_attempts ]; do
    # 检查常见的前端端口
    if check_port 5173 || check_port 8082 || check_port 8083; then
        # 从日志中获取实际端口
        local actual_port=$(grep -o "http://localhost:[0-9]*" "$FRONTEND_LOG" | head -1 | grep -o "[0-9]*$")
        if [ -n "$actual_port" ]; then
            print_message $GREEN "前端服务已在端口 $actual_port 启动成功！"
            frontend_started=true
            break
        fi
    fi
    # ...
done
```

**Node.js脚本 (`start.js`) 改进**：
```javascript
// 修复前：只等待端口 5173
if (!await waitForPort(5173, '前端服务')) {
    // 失败处理
}

// 修复后：智能检测多个端口
const ports = [5173, 8082, 8083, 8084];
for (const port of ports) {
    if (await checkPort(port)) {
        const logContent = fs.readFileSync(FRONTEND_LOG, 'utf8');
        if (logContent.includes(`localhost:${port}`)) {
            Logger.success(`前端服务已在端口 ${port} 启动成功！`);
            frontendStarted = true;
            break;
        }
    }
}
```

### 3. **改进状态显示**

**动态端口检测**：
```bash
# 检测实际运行的前端端口
for port in 5173 8082 8083 8084; do
    if check_port $port; then
        if grep -q "localhost:$port" "$FRONTEND_LOG" 2>/dev/null; then
            frontend_port=$port
            frontend_running=true
            break
        fi
    fi
done

if [ "$frontend_running" = true ]; then
    print_message $GREEN "✅ 前端服务: 运行中 (http://localhost:$frontend_port)"
else
    print_message $RED "❌ 前端服务: 未运行"
fi
```
