## 🔄 密码重置

如果忘记密码，可以通过以下方式重置：

### 方法1: 使用API重置（需要系统访问权限）
```bash
# 为用户ID 7重置密码
curl -X PUT http://localhost:8080/api/v1/users/7 \
  -H "Content-Type: application/json" \
  -d '{"password": "新密码"}'
```

### 方法2: 数据库直接操作（需要数据库访问权限）
```sql
-- 更新密码哈希（需要先用bcrypt加密新密码）
UPDATE users SET password = '加密后的密码哈希' WHERE id = 7;
```
