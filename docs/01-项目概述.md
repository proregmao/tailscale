# 01-Tailscale项目概述

## 项目简介

Tailscale是一个基于WireGuard®的私有网络解决方案，旨在让私有WireGuard网络变得简单易用。它是一个开源项目，提供了安全、简单的点对点网络连接。

## 核心理念

- **简化WireGuard**: 让复杂的WireGuard配置变得简单
- **零配置网络**: 自动处理NAT穿越、密钥管理和网络发现
- **安全第一**: 基于现代加密技术，端到端加密
- **跨平台支持**: 支持多种操作系统和设备

## 项目结构

### 主要组件

1. **tailscaled** - 核心守护进程
   - 运行在Linux、Windows、macOS、FreeBSD、OpenBSD
   - 处理网络连接和路由
   - 管理WireGuard隧道

2. **tailscale** - 命令行客户端
   - 用户交互界面
   - 配置和管理工具
   - 状态查询和诊断

3. **控制平面** - 协调服务
   - 节点发现和认证
   - 密钥分发
   - 网络拓扑管理

### 技术架构

```
┌─────────────────┐    ┌─────────────────┐
│   tailscale     │    │   tailscaled    │
│   (CLI客户端)    │◄──►│   (守护进程)     │
└─────────────────┘    └─────────────────┘
                              │

                              │

                    └─────────────────┘
```

## 开源策略

- **核心代码开源**: 包含大部分功能的源代码
- **移动端GUI闭源**: iOS和Android的GUI部分不开源
- **商业服务**: 提供托管的控制平面服务

## 构建要求

- **Go版本**: 需要最新的Go版本（当前为Go 1.23）
- **自定义Go**: 使用Tailscale的Go分支构建发布版本
- **跨平台**: 支持多种操作系统和架构

## 安装方式

1. **包管理器**: 通过各发行版的包管理器
2. **官方包**: https://pkgs.tailscale.com
3. **源码编译**: 从源码构建
